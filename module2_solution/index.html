It was designed to be compiled to provide low-level access to memory and language constructs that map efficiently to machine instructions, all with minimum runtime support.

C++

C++ is a general purpose programming language created by Bjarne Stroustrup as an extension of the C programming language, or "C with Classes". It was designed witj abias toward system programming and embedded, resource constrained softwware and large systems, with performance, efficiency, and flexibility of use as its design highlights.

JAVA

Java is a general purpose programming language that is class biased, object oriented, and designed to have as few implementation dependencies as possible. Java is used to create complete applications that may run on a single computer or be distributed among servers and clients in a network as well as usedto build a small application module or applet.
